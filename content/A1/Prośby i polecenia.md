# Прохання і прості вказівки — просьбы и команды — requests and simple commands

На A1 корисно вміти ввічливо попросити й дати просте прохання/інструкцію: *Proszę…*, *Proszę + bezokolicznik*.

## Proszę… / Будь ласка… / Please…

**Proszę** використовується дуже широко: як «будь ласка», «прошу», «ось».

Для прохання: **Proszę mówić wolniej.** (Говоріть повільніше, будь ласка.)

## Polecenia / Вказівки / Commands

У сервісі часто чуєш прості вказівки: **Proszę poczekać**, **Proszę podpisać**.

## Słownictwo / Словник / Vocabulary

**proszę** — будь ласка / пожалуйста / please / here you go
**poczekać** — почекати / подождать / to wait
**powtórzyć** — повторити / повторить / to repeat
**mówić wolniej** — говорити повільніше / говорить медленнее / speak slower
**wejść** — увійти / войти / to enter
**usiąść** — сісти / сесть / to sit

## Przykłady / Приклади / Examples

- Proszę powtórzyć. — Повторіть, будь ласка. — Please repeat.
- Proszę mówić wolniej. — Говоріть повільніше, будь ласка. — Please speak slower.
- Proszę poczekać. — Почекайте, будь ласка. — Please wait.
- Proszę wejść. — Заходьте, будь ласка. — Please come in.
- Proszę usiąść. — Сідайте, будь ласка. — Please sit down.

> [!IMPORTANT]
> У Польщі **proszę** — дуже універсальне слово. Воно звучить ввічливо майже в будь-якій ситуації.
