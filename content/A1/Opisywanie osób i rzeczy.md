# Опис людей і речей — описание людей и вещей — describing people and things

Тут — базові прикметники та конструкції для опису: зовнішність, предмети, кімната, місто. На A1 важлива простота.

## Jaki? Jaka? Jakie? / Який? Яка? Які? / What is it like?

Питання **jaki/jaka/jakie** допомагає просити опис.

Відповідь: **To jest + przymiotnik + rzeczownik** (Це + прикметник + іменник).

## Bardzo / Trochę / Дуже / Трохи

Для інтенсивності: **bardzo** (дуже), **trochę** (трохи), **nie bardzo** (не дуже).

## Słownictwo / Словник / Vocabulary

**ładny** — гарний / красивый / nice/pretty
**brzydki** — некрасивий / некрасивый / ugly
**duży** — великий / большой / big
**mały** — малий / маленький / small
**nowy** — новий / новый / new
**stary** — старий / старый / old

## Przykłady / Приклади / Examples

- To jest duży dom. — Це великий будинок. — This is a big house.
- Mój pokój jest mały. — Моя кімната маленька. — My room is small.
- To jest bardzo ładny samochód. — Це дуже гарна машина. — This is a very nice car.
- On jest wysoki. — Він високий. — He is tall.
- Ona jest sympatyczna. — Вона приємна. — She is nice.
