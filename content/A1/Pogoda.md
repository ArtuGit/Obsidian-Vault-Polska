# Погода — погода — weather

Погода — часта тема small talk. На A1 достатньо знати кілька прикметників і конструкцію **Jest…** / **Pada…**.

## Jak jest? / Яка погода? / What’s the weather like?

Найпростіше: **Jest + przymiotnik**: *Jest zimno / ciepło / słonecznie*.

Коли йде дощ/сніг: **Pada deszcz / Pada śnieg**.

## Rozmowa / Розмова / Small talk

Короткі реакції: **Super!**, **Szkoda…**, **Nie lubię zimy.**

## Słownictwo / Словник / Vocabulary

**pogoda** — погода / погода / weather
**zimno** — холодно / холодно / cold
**ciepło** — тепло / тепло / warm
**słonecznie** — сонячно / солнечно / sunny
**pada deszcz** — йде дощ / идёт дождь / it’s raining
**wieje wiatr** — дує вітер / дует ветер / it’s windy

## Przykłady / Приклади / Examples

- Dzisiaj jest zimno. — Сьогодні холодно. — Today it’s cold.
- Jest słonecznie. — Сонячно. — It’s sunny.
- Pada deszcz. — Йде дощ. — It’s raining.
- Wieje wiatr. — Дме вітер. — It’s windy.
- Jaka jest pogoda jutro? — Яка погода завтра? — What’s the weather tomorrow?

> [!TIP]
> Запам’ятай два шаблони: **Jest…** (стан) і **Pada…** (опади).
