# Прикметник (основи) — прилагательное (основы) — adjectives (basics)

Прикметники описують предмети та людей: *duży, mały, dobry, zły, ładny*. На A1 головне — знати базову лексику й вміти сказати просте речення.

## Jaki? / Який? / What kind?

Питання: **Jaki? Jaka? Jakie?**

Відповідь: **To jest + przymiotnik + rzeczownik**.

## Bardzo / Дуже / Very

Підсилення: **bardzo** (дуже), **trochę** (трохи).

## Słownictwo / Словник / Vocabulary

**dobry** — добрий / хороший / good
**zły** — поганий / плохой / bad
**ładny** — гарний / красивый / nice
**brzydki** — некрасивий / некрасивый / ugly
**duży** — великий / большой / big
**mały** — малий / маленький / small

## Przykłady / Приклади / Examples

- To jest dobry film. — Це хороший фільм. — This is a good movie.
- To jest bardzo dobra kawa. — Це дуже добра кава. — This is very good coffee.
- Mój pokój jest mały. — Моя кімната маленька. — My room is small.
- To jest duży dom. — Це великий будинок. — This is a big house.
- To nie jest zły pomysł. — Це не погана ідея. — That’s not a bad idea.
