# Дні тижня — дни недели — days of the week

На A1 ти дуже часто кажеш: **сьогодні/завтра який день?** і **коли?**

## Nazwy dni / Назви днів / Day names

**dzień tygodnia** — день тижня / день недели / day of the week

- **poniedziałek** — понеділок / понедельник / Monday
- **wtorek** — вівторок / вторник / Tuesday
- **środa** — середа / среда / Wednesday
- **czwartek** — четвер / четверг / Thursday
- **piątek** — п’ятниця / пятница / Friday
- **sobota** — субота / суббота / Saturday
- **niedziela** — неділя / воскресенье / Sunday

> [!TIP]
> Гарна практика: повторюй дні як “пісеньку” — так швидше запам’ятовуються.

## Dziś / jutro / wczoraj / Сьогодні / завтра / вчора / Today / tomorrow / yesterday

**dziś (dzisiaj)** — сьогодні / сегодня / today

**jutro** — завтра / завтра / tomorrow

**wczoraj** — вчора / вчера / yesterday

## Przykłady (A1) / Приклади (A1) / Examples (A1)

- Dziś jest poniedziałek. — Сьогодні понеділок. — Today is Monday.
- Jutro jest wtorek. — Завтра вівторок. — Tomorrow is Tuesday.
- Wczoraj była niedziela. — Вчора була неділя. — Yesterday was Sunday.
