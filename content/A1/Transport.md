## Komunikacja / Транспорт / Public transport

Питання: **Gdzie jest przystanek?** (Де зупинка?)

## Kierunek / Напрямок / Direction

Корисно: **Dokąd jedzie ten autobus?** (Куди їде цей автобус?)

**tramwaj** — трамвай / трамвай / tram
**pociąg** — поїзд / поезд / train
**bilet** — квиток / билет / ticket
**przystanek** — зупинка / остановка / stop
**dworzec** — вокзал / вокзал / station

## Przykłady / Приклади / Examples

- Gdzie jest przystanek? — Де зупинка? — Where is the stop?
