# Кольори — цвета — colors

На A1 вистачить: назвати колір, запитати “якого кольору?”, описати прості речі (сорочка, авто, будинок).

## Kolory podstawowe / Основні кольори / Basic colors

**kolor** — колір / цвет / color

- **biały** — білий / белый / white
- **czarny** — чорний / чёрный / black
- **czerwony** — червоний / красный / red
- **niebieski** — синій / синий / blue
- **zielony** — зелений / зелёный / green
- **żółty** — жовтий / жёлтый / yellow
- **szary** — сірий / серый / gray

> [!TIP]
> Повторюй колір + предмет: *czerwone jabłko* (червоне яблуко).

## Pytanie o kolor / Питання про колір / Asking about color

- **Jaki to jest kolor?** — Який це колір? — What color is it?
- **Jakiego to jest koloru?** — Якого це кольору? — What color is it?

## Przykłady / Приклади / Examples

- To jest czerwony samochód. — Це червона машина. — This is a red car.
- Mam czarną kurtkę. — У мене чорна куртка. — I have a black jacket.
- Lubię niebieski kolor. — Я люблю синій колір. — I like the color blue.
- Ten dom jest biały. — Цей будинок білий. — This house is white.

> [!NOTE]
> Форми прикметника змінюються (np. *czarny / czarna / czarne*). На A1 головне — щоб було зрозуміло.
