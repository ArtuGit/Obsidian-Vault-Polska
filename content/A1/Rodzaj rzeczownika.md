# Рід іменника — род существительного — noun gender

У польській іменники мають рід: **męski** (чол.), **żeński** (жін.), **nijaki** (сер.). Це важливо для прикметників і займенників.

## Jak rozpoznać? / Як розпізнати? / How to recognize?

Дуже грубо: **-a** часто жіночий (kawa), але є винятки.

Середній рід часто закінчується на **-o/-e** (okno, morze).

## Po co? / Навіщо? / Why it matters

Рід впливає на форми: **ten/ta/to** та прикметники: **dobry/dobra/dobre**.

## Słownictwo / Словник / Vocabulary

**rodzaj** — рід / род / gender
**męski** — чоловічий / мужской / masculine
**żeński** — жіночий / женский / feminine
**nijaki** — середній / средний / neuter
**ten/ta/to** — цей/ця/це / этот/эта/это / this
**dobry/dobra/dobre** — добрий/добра/добре / хороший/хорошая/хорошее / good

## Przykłady / Приклади / Examples

- To jest ten dom. — Це цей будинок. — This is this house.
- To jest ta kawa. — Це ця кава. — This is this coffee.
- To jest to okno. — Це це вікно. — This is this window.
- To jest dobry film. — Це хороший фільм. — This is a good movie.
- To jest dobra kawa. — Це добра кава. — This is good coffee.

> На A1 достатньо вивчити **ten/ta/to** і кілька прикладів — решта прийде з практикою.
