# Однина і множина — единственное и множественное — singular & plural

На A1 достатньо: розрізняти “один/багато” і вміти сказати прості фрази типу “це книга / це книги”.

## Słowa podstawowe / Базові слова / Basic words

**liczba pojedyncza** — однина / единственное число / singular

**liczba mnoga** — множина / множественное число / plural

**jeden** — один / один / one

**dwa / trzy / cztery** — два / три / чотири / два / три / четыре / two / three / four

Powiązano:
- [[A1/Liczby 0-100]] — числа / числа / numbers

## Przykład (A1) / Приклад / Example

**książka** — книга / книга / book

- To jest książka. — Це книга. — This is a book.
- To są książki. — Це книги. — These are books.

**jabłko** — яблуко / яблоко / apple

- To jest jabłko. — Це яблуко. — This is an apple.
- To są jabłka. — Це яблуka. — These are apples.

> [!NOTE]
> На A1 не потрібно одразу вчитись усіх закінчень. Вчись на прикладах і поступово додавай нові слова.

## Przydatne zdania / Корисні речення / Useful sentences

- Mam jeden zeszyt. — У мене один зошит. — I have one notebook.
- Mam dwa zeszyty. — У мене два зошити. — I have two notebooks.
- Nie mam pieniędzy. — У мене немає грошей. — I don’t have money.
