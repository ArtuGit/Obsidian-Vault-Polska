# Ступені прикметників — степени прилагательных — adjective comparison

На рівні A1 досить знати найчастіші форми: *duży–większy*, *mały–mniejszy*, *dobry–lepszy*, *zły–gorszy*.

## Stopień wyższy / Вищий ступінь / Comparative

Порівняння: **X jest większy niż Y** (X більший, ніж Y).

Інколи використовують **bardziej** (більш) з довшими словами: **bardziej interesujący**.

## Naj- / Най- / Superlative

Найвищий ступінь часто з **naj-**: **największy**, **najlepszy**.

## Słownictwo / Словник / Vocabulary

**duży** — великий / большой / big
**większy** — більший / больше / bigger
**mały** — малий / маленький / small
**mniejszy** — менший / меньше / smaller
**dobry** — добрий / хороший / good
**lepszy** — кращий / лучше / better

## Przykłady / Приклади / Examples

- Ten pokój jest większy niż tamten. — Ця кімната більша, ніж та. — This room is bigger than that one.
- To jest lepsza kawa. — Це краща кава. — This is better coffee.
- Dzisiaj jest cieplej niż wczoraj. — Сьогодні тепліше, ніж учора. — Today is warmer than yesterday.
- To jest najprostsze zadanie. — Це найпростіше завдання. — This is the easiest task.
- Mój telefon jest mniejszy. — Мій телефон менший. — My phone is smaller.

> [!TIP]
> Найчастіше просто вивчи кілька нерегулярних пар: **dobry–lepszy**, **zły–gorszy**, **duży–większy**, **mały–mniejszy**.
