# Заперечення «nie» — отрицание «nie» — negation “nie”

У польській мові заперечення робиться словом **nie**. На A1 достатньо вміти сказати: *nie mam*, *nie wiem*, *nie rozumiem*.

## Jak negować? / Як заперечувати? / How to negate?

Став **nie** перед дієсловом: **Nie wiem**, **Nie mam czasu**, **Nie rozumiem**.

У короткій відповіді можна сказати просто: **Nie**.

## Częste gotowce / Часті готові фрази / Common phrases

Вивчи кілька готових фраз — вони дуже часто зустрічаються у розмові.

## Słownictwo / Словник / Vocabulary

**nie** — ні / не / нет / не / no / not
**nie wiem** — не знаю / не знаю / I don’t know
**nie rozumiem** — не розумію / не понимаю / I don’t understand
**nie mam** — у мене немає / у меня нет / I don’t have
**nie mogę** — не можу / не могу / I can’t
**nie lubię** — не люблю / не люблю / I don’t like

## Przykłady / Приклади / Examples

- Nie wiem. — Я не знаю. — I don’t know.
- Nie rozumiem po polsku. — Я не розумію польською. — I don’t understand in Polish.
- Nie mam czasu. — У мене немає часу. — I don’t have time.
- Nie mogę dzisiaj. — Я не можу сьогодні. — I can’t today.
- Nie lubię kawy. — Я не люблю каву. — I don’t like coffee.

> [!TIP]
> Суперкорисні фрази: **Nie wiem**, **Nie rozumiem**, **Nie mam** — вони закривають багато ситуацій на A1.
