# Вказівні займенники — указательные местоимения — demonstratives

Вказівні слова допомагають показати на предмет: **ten/ta/to** (цей/ця/це), **tamten/tamta/tamto** (той/та/те).

## Ten / Ta / To / Цей / Ця / Це / This

На A1 достатньо вміти сказати: **ten dom**, **ta kawa**, **to okno**.

## Tamten… / Той… / That

Для «той, що там» часто: **tamten/tamta/tamto**.

## Słownictwo / Словник / Vocabulary

**ten** — цей / этот / this (m.)
**ta** — ця / эта / this (f.)
**to** — це / это / this (n.)
**tamten** — той / тот / that (m.)
**tamta** — та / та / that (f.)
**tamto** — те / то / that (n.)

## Przykłady / Приклади / Examples

- To jest ten dom. — Це цей будинок. — This is this house.
- To jest ta kawa. — Це ця кава. — This is this coffee.
- To jest to okno. — Це це вікно. — This is this window.
- Tamten dom jest duży. — Той будинок великий. — That house is big.
- Tamta książka jest dobra. — Та книжка добра. — That book is good.

> [!TIP]
> Практика: покажи на предмет і скажи **ten/ta/to** + слово — це швидко автоматизується.
