# Місяці й пори року — месяцы и времена года — months & seasons

На A1 потрібно: назвати місяць, пору року і сказати просту дату.

## Pory roku / Пори року / Seasons

**pora roku** — пора року / время года / season

- **wiosna** — весна / весна / spring
- **lato** — літо / лето / summer
- **jesień** — осінь / осень / autumn
- **zima** — зима / зима / winter

## Miesiące (wybrane) / Місяці (вибрані) / Months (selected)

**miesiąc** — місяць / месяц / month

- **styczeń** — січень / январь / January
- **marzec** — березень / март / March
- **maj** — травень / май / May
- **lipiec** — липень / июль / July
- **wrzesień** — вересень / сентябрь / September
- **grudzień** — грудень / декабрь / December

> [!TIP]
> Не потрібно вчитись одразу всі 12. Почни з 4–6 найчастіших (твій день народження, теперішній місяць).

## Przykłady / Приклади / Examples

- Jest zima. — Зараз зима. — It’s winter.
- Lubię lato. — Я люблю літо. — I like summer.
- Mam urodziny w maju. — У мене день народження в травні. — My birthday is in May.
- Dziś jest 5 grudnia. — Сьогодні 5 грудня. — Today is December 5th.
