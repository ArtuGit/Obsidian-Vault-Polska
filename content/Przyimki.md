# Прийменники / Предлоги / Prepositions

**Przyimek** (прийменник / предлог / preposition) — це незмінна частина мови, яка вказує на відношення між словами у реченні. У польській мові прийменники керують [[Przypadki|відмінками]] іменників.

---

## Прийменники з родовим відмінком / Prepositions with Genitive (Dopełniacz)

| Przyimek | Переклад (UA/EN) | Приклад |
|----------|------------------|---------|
| **do** | до / to | Idę **do** sklepu. — Іду до магазину. / I'm going to the store. |
| **z/ze** | з, із / from, out of | Jestem **z** Polski. — Я з Польщі. / I'm from Poland. |
| **od** | від / from | List **od** mamy. — Лист від мами. / A letter from mom. |
| **bez** | без / without | Kawa **bez** cukru. — Кава без цукру. / Coffee without sugar. |
| **dla** | для / for | Prezent **dla** ciebie. — Подарунок для тебе. / A gift for you. |
| **u** | у, в (когось) / at (someone's place) | Byłem **u** lekarza. — Був у лікаря. / I was at the doctor's. |
| **obok** | поруч, біля / next to | Stoję **obok** domu. — Стою поруч із домом. / I'm standing next to the house. |
| **koło** | біля, коло / near | Mieszkam **koło** parku. — Живу біля парку. / I live near the park. |
| **blisko** | близько / close to | **Blisko** centrum. — Близько центру. / Close to the center. |
| **daleko** | далеко від / far from | **Daleko** od miasta. — Далеко від міста. / Far from the city. |
| **podczas** | під час / during | **Podczas** wakacji. — Під час канікул. / During vacation. |
| **wśród** | серед / among | **Wśród** przyjaciół. — Серед друзів. / Among friends. |
| **zamiast** | замість / instead of | **Zamiast** herbaty. — Замість чаю. / Instead of tea. |
| **oprócz** | крім / except, besides | Wszyscy **oprócz** mnie. — Всі, крім мене. / Everyone except me. |
| **według** | згідно з, за / according to | **Według** planu. — Згідно з планом. / According to the plan. |

## Прийменники руху та місця / Movement vs. Location Prepositions

Деякі прийменники змінюють відмінок залежно від того, чи описують **рух** (напрямок) чи **місце** (локацію):

| Przyimek | Рух (Biernik) | Місце (Narzędnik/Miejscownik) |
|----------|---------------|-------------------------------|
| **na** | Idę **na** pocztę. (Іду на пошту) | Jestem **na** poczcie. (Я на пошті) |
| **w** | Idę **w** las. (Іду в ліс) | Jestem **w** lesie. (Я в лісі) |
| **nad** | Jadę **nad** morze. (Їду на море) | Jestem **nad** morzem. (Я на морі) |
| **pod** | Kładę **pod** stół. (Кладу під стіл) | Leży **pod** stołem. (Лежить під столом) |
| **za** | Idę **za** dom. (Іду за дім) | Stoję **za** domem. (Стою за домом) |
| **przed** | Stawiam **przed** dom. (Ставлю перед дім) | Stoi **przed** domem. (Стоїть перед домом) |
| **między** | Wchodzę **między** ludzi. (Заходжу між людей) | Stoję **między** ludźmi. (Стою між людьми) |

## Часові прийменники / Time Prepositions

| Przyimek | Використання | Приклад |
|----------|-------------|---------|
| **o** + Loc. | година / hour | **O** ósmej. — О восьмій. / At eight o'clock. |
| **w** + Acc. | дні тижня / days | **W** poniedziałek. — У понеділок. / On Monday. |
| **w** + Loc. | місяці, роки / months, years | **W** styczniu. — У січні. / In January. |
| **przed** + Instr. | до (часу) / before | **Przed** obiadem. — Перед обідом. / Before lunch. |
| **po** + Loc. | після / after | **Po** obiedzie. — Після обіду. / After lunch. |
| **za** + Acc. | через (час) / in (time) | **Za** godzinę. — Через годину. / In an hour. |
| **od... do** | від... до / from... to | **Od** rana **do** wieczora. — З ранку до вечора. / From morning to evening. |

## Примітки / Notes

> [!tip] Порада / Tip
> Прийменники **z** і **ze**, **w** і **we** — це варіанти одного прийменника. Форми **ze** і **we** використовуються перед словами, що починаються зі збігу приголосних:
> - **ze** szkołą (зі школою)
> - **we** wtorek (у вівторок)
> - **ze** mną (зі мною)

> [!warning] Увага / Attention  
> Прийменник **z** може вимагати різних відмінків залежно від значення:
> - **z** + Dopełniacz (з, від): Jestem **z** Ukrainy. — Я з України.
> - **z** + Narzędnik (разом з): Idę **z** bratem. — Іду з братом.
