# План навчання A1 — план обучения A1 — A1 learning plan

Це **дорожня карта рівня A1** для польської (język polski) з посиланнями на теми.
Кожне посилання має короткий переклад українською / російською / англійською (текст **поза** посиланням).

> [!TIP]
> Режим: 20–40 хв щодня. Спочатку **вимова + базові фрази**, потім **дієслова і відмінки**, далі — лексика і практика.

## Як працювати з планом / How to use the plan

- Читай теми **зверху вниз** (від простого до складного).
- В кожній темі додай власні приклади: *PL → UA/EN*.
- Повертайся до довідників:

  - [[Czesci mowy]] — частини мови / части речи / parts of speech
  - [[Rodzaj rzeczowników i przymiotników]] — рід / род / gender
  - [[Przypadki]] — відмінки / падежи / cases
  - [[Przyimki]] — прийменники / предлоги / prepositions
  - [[Zaimki wskazujące]] — вказівні займенники / указательные / demonstratives

> [!IMPORTANT]
> На A1 достатньо добре відчути **Mianownik, Biernik, Dopełniacz** (називний, знахідний, родовий) і найчастіші прийменники.
